# MagicShock

Developed with Unreal Engine 4

# Project Features
* First-person Character Movement
* Character Skill System
    * Explosive Projectile Attack (Hits a lot of pawns)
    * Magic Projectile
* HUD
* Attribute Component
* AI
    * Minion AI with Behavior Trees
